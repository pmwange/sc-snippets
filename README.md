# Sugar Snippets

WordPress plugin: add custom PHP snippets (like functions.php). Enable or disable each snippet from **Settings → Sugar Snippets**.

## Usage

1. Install in `wp-content/plugins/sc-snippets/` and activate in WordPress.
2. Go to **Settings → Sugar Snippets** to choose which snippets are active.
3. Edit `snippets.php` or add `.php` files in `snippets/` — only enabled snippets are loaded.
