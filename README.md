# sc-snippets

WordPress plugin: add custom PHP snippets (like functions.php). When active, all snippets in `snippets.php` and the `snippets/` folder are loaded.

## Usage

1. Install in `wp-content/plugins/sc-snippets/` and activate in WordPress.
2. Edit `snippets.php` or add `.php` files in `snippets/` — they run when the plugin is active.
